S'identifier via un jeton (token) anonymisé (OPcode)

OPcode {{randomToken}}
Ce jeton (OPcode) est votre identifiant. Il est unique et généré de manière aléatoire.

 

Vous êtes le seul détenteur de l'association entre vous et ce jeton. Si vous souhaitez questionner l'équipe de recherche à propos de vos données, ce jeton vous sera demandé.
Si vous disposez d'un jeton (OPcode) d'une installation précédente, vous pouvez l'utiliser ici pour conserver le même compte. Notez qu'il n'y a pas de jetons incorrects. Si vous entrez un OPcode qui ne correspond pas à un code existant, un nouveau compte sera créé.
